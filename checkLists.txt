Позитивные
	- Проверка возможности объявления класса >> при создании объекта заданного класса, получен объект с требуемым классом без возникновения ошибок
	- Проверка результата при вызове метода класса >> требуется создать объект заданного класса и вызвать заданный метод. В результате должен быть получен объект окна изображения с соответствующими диаграммами и данными
	- Проверка возможности изменения атрибутов класса >> требуется создать объект заданного класса и перезаписать значение атрибута. В результате исходное значение атрибута и сохраненное должны отличаться.

Компонентные 
	- Проверка возможности изменения атрибутов класса до и после создания объекта с заданного класса;
	- Результатом вызова метода plotVelocityAndTrajectory класса является объект типа figure;
	- Результатом вызова метода solveDiffEquation класса является вектор с соответствующими данными (координата радиуса, координата угла, производная координаты радиуса, производная координаты угла, линеная скорость, время);

Негативные
	- Проверка отказаустойчивости методов класса при явно невозможных значениях атрибутов (кол-во периодов, параметры спутника, масса планеты меньше или равно нулю; начальная высота орбиты < 0)
	- Проверка отказоустойчивости методов класса при создании новых атрибутов класса (создание объекта заданного класса с новыми атрибутами не перечисленными в разделе properties)
